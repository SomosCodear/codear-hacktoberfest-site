Te damos la bienvenida a Hacktoberfest®.

¿De qué se trata? Es un evento organizado por DigitalOcean, en el que durante todo Octubre, celebramos y apoyamos a la comunidad del software de código abierto.

Este año, CoDeAr se suma como organizadora de un evento regional para alentar y fomentar la participación de personas de todas las disciplinas allegadas a tecnología para ayudar a mantener proyectos de código abierto.
